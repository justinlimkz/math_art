Mandelbrot set

This fractal was discovered by mathematician Benoit Mandelbrot. It looks very complex (ha!)! But it is simply the set of all complex z such that the sequence

z_0 = z, z_{n+1} = z_{n}^2+z, n = 0, 1, ...

is bounded as n approaches infinity.

Fibonacci approximation to the "Golden spiral"

This is not a fractal - for the simple reason that Fibonacci numbers are integers! It terminates somewhere in the middle.

However, it does approximate the "Golden spiral", because the ratio between consecutive Fibonacci numbers converges to the golden ratio.
